# User API

## 1.Get Users

### Method:GET

### API endpoint:

https://website.com/BookingHotel/users


````ts
export interface Users {
    status: 'error' | 'success'
    msg: string
    data: Data[]
    code: number
}

export interface Data {
    ID: string
    hoTen: string
    username: string
    email: string
    role: string
    urlQRcode:string
    createDate: string
}
````

## 3.Update user

### Method:PUT

### API endpoint:

https://website.com/BookingHotel/users


##### data x-www-form-urlencoded

key | type | value
--- | --- | ---
ID | string | id Của user
token | string | code xác thực bắn trên header
?username | string | tên tài khoản
?role | string | quyền
?email | string | email
?hoTen | string | ten
?diaChi | string | địa chỉ
?ngaySinh | string | Ngày Sinh


````ts
export interface User {
    status: 'error' | 'success'
    msg: string
    data: []
    code: number
}
````
